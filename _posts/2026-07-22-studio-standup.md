# This cycle in the studio

**Pulse:** a new public project went live and existing builds kept moving —
with some housekeeping on the fleet under way.

This cycle the studio stood up a new public dataset, **canonical-castles**, which
explores where different national castle registers disagree with each other —
and is honest, in its own issues, about the judgment work still ahead. Work on
the multiplayer text-world rebuild advanced into its measurement phase, capturing
how much of the historic command set is genuinely covered. The studio also
published its latest cost report to the public ledger, keeping the running record
of what the machinery spends. Alongside the new work, the team spent the cycle on
reliability housekeeping — chasing down flaky checks and a few worker runs that
need tuning — so the shipped surfaces stay honest about their own health.
