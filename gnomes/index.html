A gnome is a single agent that handles one task end to end. Platform gnomes serve the studio itself; project gnomes serve one or more projects. Gnomes work in knolls — teams with a shared purpose and a shared knowledge base that every member carries into its runs.

Each portrait is the gnome at the craft-ladder stage it has earned in the ledger — a novice's tool is plain and held with care; a meister's is a masterwork.
